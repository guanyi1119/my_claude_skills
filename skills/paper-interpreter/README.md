# 论文解读 Skill (paper-interpreter)

一个用于深度解读学术论文的skill，支持中英文论文和PDF/文本格式，包含论文归档管理功能。

## 功能特性

- 📄 **多格式支持**：支持PDF文件和纯文本输入
- 🌐 **中英文论文**：自动识别并处理中英文论文
- 📊 **结构化分析**：提取贡献点、方法论、实验结果等关键信息
- 🔬 **消融实验分析**：专门分析论文中的消融实验
- 💡 **术语解释**：自动解释专业术语
- 🎯 **交互式输出**：可展开的详细分析结构
- 📚 **论文归档管理**：
  - 可选择归档或不归档论文
  - 按标签分类保存
  - Excel文件管理，支持超链接直接打开
  - 自动重命名和文件管理


## 安装依赖

如需使用PDF解析和归档功能，请安装以下库：

```bash
pip install pdfplumber openpyxl
```

## 使用方法

### 基本使用

当用户要求解读论文时，skill会自动触发。你可以：

1. **直接粘贴论文文本**：
   ```
   请解读这篇论文：[粘贴论文内容]
   ```

2. **提供PDF文件路径**：
   ```
   请解读这个PDF论文：path/to/paper.pdf
   ```

3. **询问特定问题**：
   ```
   这篇论文的主要贡献点是什么？
   ```

### 输出结构

分析结果会以交互式格式呈现：

- 🎯 **快速摘要**：一句话总结
- 📋 **详细分析**（可展开）：
  - 研究背景与问题
  - 主要贡献点
  - 方法论细节
  - 实验设置
  - 消融实验分析
  - 主要结果
  - 结论与未来工作
  - 关键术语解释
- 📚 **归档询问**：询问是否需要归档论文

### 归档流程

在解析完成后，skill会询问是否需要归档论文：

#### 如果不归档
- 删除临时生成的PDF和MD文件
- 删除临时提取的文本文件
- 结束流程

#### 如果归档
1. **输入归档原因**：描述为什么要归档这篇论文
2. **输入归档标签**：用于分类（如"计算机视觉"、"自然语言处理"等）
3. **自动处理**：
   - 创建/更新Excel归档文件
   - 重命名并保存文件到对应目录
   - 添加超链接到Excel文件

### 归档文件位置

- **Excel文件**：`d:\Workspace\PaperReader\paper-interpreter\archive\paper_archive.xlsx`
- **PDF文件**：`d:\Workspace\PaperReader\paper-interpreter\archive\pdf\`（重命名后的）
- **解读报告**：`d:\Workspace\PaperReader\paper-interpreter\archive\md\`（重命名后的）

### Excel文件结构

每个标签对应一个Sheet页，包含以下列：
- **标题**：论文标题
- **年份**：发表年份
- **作者单位**：作者所属机构
- **收录会议/杂志**：CVPR、ICML、IEEE等
- **关键词**：关键词（逗号分隔）
- **归档原因**：用户输入的归档原因
- **归档日期**：归档时间戳
- **PDF路径**：本地PDF文件路径（超链接）
- **解读报告路径**：本地MD文件路径（超链接）

## 主要功能说明

### 1. 论文概览

快速提取论文的基本信息，包括标题、作者、发表信息等。

### 2. 贡献点提取

重点突出论文的3-5个主要贡献，每个贡献包含：
- 具体描述
- 创新点
- 重要性

### 3. 方法论分析

详细分析技术方法：
- 核心算法/模型架构
- 关键技术步骤
- 数学公式
- 实现细节

### 4. 消融实验分析

专门分析论文中的消融实验：
- 实验目的
- 实验设计
- 实验结果
- 结论

### 5. 术语解释

解释论文中的3-5个重要专业术语。

## 常见问题

**Q: 支持哪些语言的论文？**

A: 主要支持中英文论文，能自动检测语言并调整分析策略。

**Q: PDF解析需要什么依赖？**

A: 需要安装 `pdfplumber` 库。

**Q: 如何提高分析准确性？**

A: 提供完整的论文内容（特别是引言、方法论、实验章节）可以提高分析质量。
